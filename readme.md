LrcAPI是一个完全开源的歌词API，允许通过简易的API接口获取歌词、封面、音乐标签等信息，目前已有多个音乐项目接入。
v2版本使用FastAPI进行完全重构，提供更加清晰的API文档和更高的性能，将会作为未来的主分支。
